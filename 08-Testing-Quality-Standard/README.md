# 08. 测试与质量标准 (Testing, Quality & Standard)

本模块介绍音频系统的评估体系，包括量化的客观指标测试以及各行业的合规性认证标准。

## 📖 章节导航

1.  **[音频客观测试指标 (Objective Metrics)](./01-Objective-Metrics.md)**
    *   核心指标：THD+N, SNR, 频响曲线, 动态范围。
    *   测试系统拓扑与专业测试仪器（AP, B&K）。
    *   客观指标与主观听感的权衡。

2.  **[行业通信标准与认证 (Industry Standards)](./02-Industry-Standards.md)**
    *   语音质量评估：PESQ (P.862) 与 POLQA (P.863)。
    *   车载专用标准：ITU-T P.1100/P.1110, CarPlay 认证。
    *   会议系统认证：Microsoft Teams, Zoom Certified。
    *   标准化认证测试流程。

3.  **[音频测试工具实操指南 (Testing Tools & Practice)](./03-Testing-Tools-Practice.md)**
    *   Audio Precision 专业测试系统与自动化 API。
    *   PESQ / POLQA 语音质量客观评估实操。
    *   MOS 主观评价方法（P.800 / MUSHRA）。
    *   Audacity / SoX / Python 音频分析工具链。
    *   Android 端自动化音频测试脚本。

4.  **[Android CTS/VTS 与自动化测试 (CTS/VTS & Automation)](./04-CTS-VTS-Automation.md)**
    *   CTS 音频测试模块与高频失败用例解析。
    *   VTS Audio HAL 测试：IStreamOut/IStreamIn/IEffect。
    *   Python 自动化测试框架搭建与 CI 集成。
    *   音频压力测试脚本（并发流、路由切换、资源泄漏）。
