# 01. 声学与数字音频基础 (Acoustics & Digital Audio)

本模块介绍音频技术的物理与数学基础，是所有后续高级技术（如 DSP、Android 框架、车载系统）的地基。

## 📖 章节导航

1.  **[声波物理特性 (Sound Wave Physics)](./01-Sound-Wave-Physics.md)**
    *   声音的产生、传播、频率、波长、声速与相位。
    *   声学现象：干涉、衍射、反射。
    *   分贝 (dB) 系统与 SPL 计算。

2.  **[心理声学 (Psychoacoustics)](./02-Psychoacoustics.md)**
    *   人耳听觉范围与等响曲线。
    *   频域掩蔽与时域掩蔽效应（MP3 压缩原理）。
    *   双耳定位与 HRTF。

3.  **[数字音频基础 (Digital Audio Fundamentals)](./03-Digital-Audio-Fundamentals.md)**
    *   ADC 过程：采样、量化、编码。
    *   奈奎斯特采样定理与混叠。
    *   位深、动态范围、量化噪声与抖动。
    *   PCM 数据格式详解。

4.  **[音频软件核心概念 (Audio Software Core Concepts)](./04-Audio-Software-Concepts.md)**
    *   七大核心量：Sample Rate / Bit Depth / Channel / Format / Frame / Buffer Size / Latency。
    *   Frame vs Sample 的区别与常见混淆。
    *   Buffer / Period / Period Count 的关系与代码示例。
    *   延迟组成拆解与获取方法。
    *   完整公式速查表与常见陷阱 FAQ。
